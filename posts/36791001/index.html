Katoolin工具集

发表于2020-01-07|更新于2020-01-07|网络安全
|字数总计:191|阅读时长:1分钟|阅读量:|评论数:

Katoolin是一个基于Python的工具，可以自动安装所有Kali Linux工具。不喜欢虚拟机，物理机安装又麻烦和危险，所以诞生了Katoolin这个工具集，让我们可以在其他发行版使用Kali的所有工具。

官方地址

特点

  • 可添加和移除kali的官方仓库
  • 安装需要的kali工具

依赖环境

  • Python 2.7
  • Linux (推荐Ubuntu)

安装

使用

  • 使用数字安装对应的工具
  • 0 直接安装所有kali工具
  • back 返回
  • gohome 回到主菜单

注意事项

如果要更新自己的系统，请务必记得先移除Kali的源。
