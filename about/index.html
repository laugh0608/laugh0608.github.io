我想我还没有成熟到可以原谅背叛《NANA》

有一说一，我是个fw

挂科蛮多，还自命不凡

喜爱编程，还懒得学，嫌太难

混到了大三，啥也不会，本专业学啥啥不懂，编程也只是个半吊子

不知道毕业了何去何从(照这样子，能不能毕业都还是未知数，，，)

自己的一些经历吧

我记得我是大一第二学期刚开始的时候，那个时候跟着一个学长学习 Linux，然后他有个博客，写着他平时的笔记和分享，真的是好羡慕。

那个时候我就萌发了要做自己博客的想法，当时因为刚接触编程嘛，啥也不会，百度搜啊搜，搜不出啥好结果，就想着自己写！！！

天啊，我是多么蠢才想着自己去写博客的页面，那个时候我的每一篇文章的 html 都是我手动写的，样式也是自己写，别提有多丑了，emmm，当时
的源代码我还留着呢，我去找找。

噗，简直了，羞耻，文章页面更是惨不忍睹，就不放截图了，当时还不是很懂服务器这些，就在西部数码买了一个虚拟主机，附送一个域名，然后就这样，每更新一篇文章，都要自己手动去写这篇文章的 HTML，简直累死，但是还是有成就感的。

但是我慢慢发现，为什么别人的博客都那么好看，而且总不能别人跟我一样傻傻的每篇文章写 HTML 吧，这个时候我的学长告诉我写文章可以 Markdown 啊，我就慢慢了解到了 Markdown 这个神奇的东西，在学习 Markdown 语法的过程中，我发现居然和 HTML 有一丢丢的相似之处，比如 Markdown 也可兼容 HTML 的图片引入，就去搜索如何将 Markdown 转换成 HTML，然后顺理成章的就了解到了 Hexo、WordPress这两个博客框架。

为什么用 Hexo 呢，因为当时自己特别喜欢折腾 Linux，电脑也是 Ubuntu 单系统，Hexo 命令行的操作自己还是非常喜欢的。

当时第一眼就看中了 Butterfly 这个主题，太好看了，功能也相当齐全，然后当时激动的还掏大钱买了服务器，买了域名，还备了案，舍友朋友都挺羡慕我，自己有个网站，可以发一些自己想写的想法。看着网站的访问量一天天增加，真的有一种超级棒的成就感，但是转折点就来了。

可能是自己有点小膨胀了吧，就去把网站提交百度检索，刚开始还好好的，网站可以被百度到，自己文章搜索也能搜出来，每天的访问量虽然增加了，但也在可以接受的范围之内。但是有一天！阿里云疯狂给我发邮件，有人恶意访问我的网站，导致资源请求量一天之内就达到了上限，我的 OSS 资源包瞬间欠费，还好也不是很多，吓得我赶紧就把服务器停了。

去咨询了解决方法，写了 robots.txt 规则，但是也无济于事，被逼无奈，只好关闭了网站，注销了备案。

真的不甘心，就又找度娘搜索一番，了解到了有 GitHubPage 这个可以免费部署静态站点的东东，还支持自定义域名，而且 Hexo 可以支持自动部署，就把博客放在了 GitHub 上，但是过了不久，又出现了新的问题。

当时我的博客采用的是 Gitalk 评论系统，然后某一天 GitHub 就给我发邮件，有人恶意评论，一天之内就发了几十封邮件，我都疯了，死活找不到问题在哪，然后邮件里我的 IP 地址，我家里所有联网的智能设备的 IP 都被爆出来了，我就害怕了，就想着彻底把博客关闭了算了，但是又不想放弃自己已经养成了写文章的习惯，就在 Hexo 上找了一个特别简单的主题，除了文章之外没有其他任何功能，这样我的邮箱才消停了。

就长这样子，然后就这样大概过了几个月，在今年的5 6月份吧，无意中看到我们群里有一个大佬还在开着博客，主题也是 butterfly，就有点羡慕，又想着把博客重新开回来，就又开始折腾，没想到主题已经更新的这么好看了，功能也更多了，甚至已经支持 PWA 特性。

配置好这个主题也花了好长时间，大部分都是在解决各种报错，甚至还被群里的大佬嘲讽：

哈哈，十万个为什么就是我本人了，但是我最近发现，我似乎把重心放在了折腾博客主题上，追求功能多、花里胡哨、好看，虽然实现某个功能或者主题某个配置成功生效了，自己也有成就感，但是当我没有什么东西改的时候我就会看着博客发呆，翻翻博客还有什么不满意的地方，似乎有点背离我学习的初衷了，忘记了博客最初是用来干嘛的。

我应该把一些好玩的，值得分享的技术或者经验分享到博客来，但是昨天晚上一个大佬也点醒了我，其实折腾博客又何尝不是一种学习？只要不把自己的学习落下，折腾博客何乐而不为？

确实，自己之前也特别喜欢折腾 Linux，动不动就在重装系统，哈哈，太菜了，但是真正的学习就是在折腾的过程中自己去思考，自己去尝试解决问题，自己解决不了再去想办法求助别人，我记得鸟哥也说过，最好的学习方法就是在折腾的过程中记录下自己解决问题的过程，遇到难题的时候拿出来看看，总会给自己灵感和经验。

望自己和正在看这篇文章的你一起加油，一起学习，一起向前。

还有十天就要开学了，也就大四了

在这三年里，真的是五味杂陈，大一大二的落寞，挂科，失恋，折磨的我不像个样子，甚至最绝望的时候想过自尽

大三可能是突然醒悟了吧，慢慢有了起色，参加了一个竞赛，认识了很多朋友，也被人叫了几个月的乙醇哥

有了那么一点点的成绩，也算是对得起自己大学这三年，没有全部白白浪费

这几天慢了下来，没有竞赛期间那么忙和紧张了，就把自己的一些经验沉淀下来，写了些文章

但是突然要面临秋招，自己还是挺迷茫的，不知道自己该何去何从，自己的伴侣也不知道怎么找，感觉又回到了浑浑噩噩的状态，是挺可怕的

给自己做了简历，感觉还是很满意，最起码有的写，给自己大学期间有个交代

秋招还是挺有信心拿到offer的，但是自己迷茫的是以后该怎么办，不知道往那个方向学，工作了真的就在自己的岗位上干上五年，甚至十年吗

自己该如何去提升自己，让我感到很焦虑
